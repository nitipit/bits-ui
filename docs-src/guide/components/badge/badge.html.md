## Usage
---
`<pkt-badge>` or `class="pkt-badge"`.
```html
<pkt-badge>1</pkt-badge>
<span class="pkt-badge"></span>
```

## Background Color
---
Badge background color can be changed by using packet-ui background color class.

```html
<!-- Example -->
<pkt-badge class="bg-amber">1</pkt-badge>
```
