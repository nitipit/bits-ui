Features

<Web Components/>

UI Components are based on Web Components standards which works across modern browsers, and can be used with any JavaScript library or framework that works with HTML.

CSS Framework (SASS)

Provides CSS Framework (SASS) which contains variables and modules for customization.

Icon System

With icon system and <pkt-icon> component, you can easily use icon from svg symbols with namespaces to combine variety of icon set.

Color System

Built-in color system to make and modified color scheme.
