Stylus Mixins

Using stylus function/mixin is the recommend way to change or customize tag style. Just use @require 'node_modules/bits-ui/src/stylus/_require' then all variables and functions will be ready to use.

Let's see stylus function/mixin for box-arrow

@require 'node_modules/bits-ui/src/stylus/_require'

// Stylus function for bits-box-arrow-top
bits-box-arrow-top
    box-arrow(
        $arrow=top,
        $arrow-size=7px,
        $bg-color=lighten(grey, 80%),
        $border-width=1px,
        $border-color=black,
        $radius=$border-radius)

For example, to customize bits-box-arrow-top css to change arrow size and background color.

// file: box-arrow.styl
@require 'node_modules/bits-ui/src/stylus/_require'

bits-box-arrow-top
    box-arrow(
        $arrow-size: 1em,
        $bg-color: $color-p)

Compile stylus to CSS, then load use it in web page.

$ stylus box-arrow.styl

<!-- <head> -->
<link rel="stylesheet" href="box-arrow.css">

<!-- <body> -->
